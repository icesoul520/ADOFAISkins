# ADOFAISkins
A mod that can replace the background of the ADOFAI main interface.

All code was generated by AI.

You can place videos or images in the backgrounds folder under the mod directory, and specify the background by entering the filename in the in-game mod settings interface. There are three modes to choose from.

Additionally, this mod includes menu icon replacement and in-editor font replacement. Icons can be customized in the uis folder within the mod directory, and fonts can be customized in the fonts folder, provided the font files are in bundle format.
